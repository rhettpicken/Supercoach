**Morning:**

**Afternoon:**
- [ ] Execute trades (if any)

- [ ] Set Captain
- [ ] Arrange field/bench optimally
- [ ] Set emergencies for each line

### Loophole Setup
```
IDEAL VC CANDIDATES:
- Plays Thursday or Friday night
- High ceiling player (100+ capable)
- Consistent performer

CAPTAIN BACKUP:
- Plays Saturday or Sunday
- Elite scorer (110+ average)
- Different team to VC
```

### Emergency Strategy
- Set EMG on bench for each line
- Consider playing order of benched players
- Lowest scoring emergency is always used

## Weekly Trade Timing Guide

### When to Trade IN

**Best timing:**
- Player has low score about to drop off rolling average
- Player coming back from injury (discounted)
- Before a favorable fixture run

**Avoid:**
- Right after a huge score (inflated price)
- Before a bye round
- Chasing last week's top scorer

### When to Trade OUT

**Best timing:**
- Player has high score about to drop off
- Injury announced
- Role change (reduced scoring)
- Before bye round (if needed)

**Avoid:**
- After a poor score due to tags (temporary)
- Knee-jerk to one bad week
- Burning trades for minor upgrades

## Breakeven Monitoring

### Weekly Check (FootyWire)

**Traffic Light System:**

| BE vs Average | Status | Action |
|---------------|--------|--------|
| Avg 15+ above BE | GREEN | Hold - rising |
| Avg 5-15 above BE | YELLOW | Monitor |
| Avg within 5 of BE | ORANGE | Watch closely |
| Avg below BE | RED | Consider trading |

### Price Change Formula
```
Weekly Price Change ≈ (3-week avg - BE) × $6,000

Example:
- Player BE: 80
- 3-week avg: 95
- Change: (95-80) × $6,000 = +$90,000
```

## Season Phase Strategy

### Phase 1: Cash Generation (Rounds 1-6)
**Focus:** Maximize rookie price rises
- Play all cash cows
- Trade failing rookies early
- Build bank for upgrades

### Phase 2: Upgrade Window (Rounds 7-11)
**Focus:** Convert cash to premiums
- Target premium with good fixtures
- Avoid players with early byes
- Build strongest possible 22

### Phase 3: Bye Rounds (Rounds 12-14)
**Focus:** Survive with full squads
- Plan trades around byes
- Maximize playing players each round
- Accept some point sacrifice

### Phase 4: POD Phase (Rounds 15-20)
**Focus:** Differentiate from pack
- Find underowned performers
- Take calculated risks
- Target fixture swings

### Phase 5: Finals (Rounds 21-22)
**Focus:** Optimal 22
- Minimal trades unless forced
- Set best team structure
- Nail captaincy
