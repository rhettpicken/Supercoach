## 1. Scoring System Deep Dive {#scoring-system}

### How Champion Data Calculates Points

SuperCoach uses Champion Data's proprietary algorithm - **NOT simple stat counting**.

**Key Principles:**
- **3,300 Point Cap**: Every game allocates exactly 3,300 total points across all 44 players
- **Context Matters**: A kick to a contest scores 0 points. An effective 40m+ kick scores more
- **Margin Scaling**: Stats in close games are worth MORE than in blowouts
- **Time Weighting**: Late-game clutch stats have different weighting

**Stat Values (Approximate):**
| Stat | Approx Points | Notes |
|------|---------------|-------|
| Kick (effective 40m+) | 3-4 | Ineffective = 0 |
| Handball (effective) | 2-3 | Ineffective = 0 |
| Mark | 3-4 | Contested marks higher |
| Tackle | 4-5 | High value |
| Clearance | 4-5 | Very high value |
| Goal | 8-10 | Including goal assist bonus |
| Hit Out to Advantage | 3-4 | Standard hitout ~1 |

**Implication for Player Selection:**
- Prioritize **contested players** over pure accumulators
- Midfielders with clearance access are gold
- Tackles are underrated - target high-tackle forwards

## 6. Captain & Loophole Strategy {#captaincy}

### The Loophole Explained

**Step-by-Step:**
1. Set Vice Captain (VC) to player in FIRST game of round
2. Wait for VC score
3. **If VC scores 120+:** Keep captain as normal, accept VC double
4. **If VC scores poorly:** Move Captain to NON-PLAYING player
5. Emergency on that line will score, VC gets doubled

**Loophole Decision Matrix:**

| VC Score | Action |
|----------|--------|
| < 90 | LOOPHOLE (move C to non-player) |
| 90-110 | HOLD (depends on C matchup) |
| 110+ | KEEP (take the double) |
| 130+ | CELEBRATE |

## 7. Trade Strategy {#trades}

### Trade Timing Philosophy

**2 Trades Per Week × 22 Rounds = 44 Total Trades**

| Phase | Rounds | Strategy | Trades Used |
|-------|--------|----------|-------------|
| Cash Gen | 1-6 | Maximize rookie gains | 8-10 |
| Upgrade | 7-12 | Convert cash to premiums | 10-12 |
| Byes | 12-14 | Navigate bye structure | 4-6 |
| POD | 15-20 | Find points of difference | 8-10 |
| Finals | 21-22 | Set & forget best 22 | 2-4 |

### Breakeven Trading

**Breakeven = Score needed to maintain price**

```
Trade OUT when:
- Player's 3-week avg < Breakeven
- Upcoming low-score game about to drop off
- Injury news or role change

Trade IN when:
- Player's 3-week avg > Breakeven
- Upcoming fixtures favor scoring
- Low-score game about to drop off rolling avg
```

### Golden Rules

1. **Never burn trades early** - Save minimum 2 for emergencies
2. **Don't sideways trade** - Unless 20+ point avg improvement
3. **Check Tuesday team sheets** - Before finalizing trades
4. **Rolling 3-week average** - Drives all price changes
